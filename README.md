💻 DSA Practice in JavaScript

This repository contains JavaScript solutions for common Data Structures and Algorithms (DSA) problems.
It’s a collection of practice exercises to strengthen problem-solving and coding skills.


🔹 Topics Covered

***Array manipulation

* Swap two numbers.
* Move zeros to the end.
* Reverse an array.
* Find maximum/minimum elements.
* Flatten Array.
* Remove Duplicates.
* Moving All Zeros To End.

*** String operations

* Reverse string.
* Palindrome check.
* Word Count.
* Vowels Count.
* Frequency Check.

*** Number problems

* Armstrong number.

*** Pattern problems

* Number pyramid.
* Square and triangle patterns.
